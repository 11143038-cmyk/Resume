# 自傳

我從小對科技與電腦充滿好奇，而進入資訊管理系後更進一步接觸程式設計與系統實作。在學期間，我不只學習課堂技術，也主動探索 AI 與電腦視覺領域。

### 🚀 技術學習經歷

大一開始學習 Python 與網頁基礎，透過專題與實作，不斷提升自己在資料處理與介面開發的能力。  
後來我將方向延伸到電腦視覺，例如使用 OpenCV、MediaPipe 等工具進行姿勢偵測專案。

我重視「實際完成作品」的訓練，不僅寫出程式，也會思考 UX 設計、資料庫架構與系統效能。

### 💪 個人特色與價值

我認為自己最大的優點是：
- 具備快速學習與邏輯分析能力  
- 能從需求出發獨立設計與開發功能  
- 具備有效溝通與團隊合作能力

這些能力讓我能在學校專案中擔任主開發者與協調者角色。
