## 📂 專案經驗 Projects

### 🔹 AI 智慧運動姿勢矯正系統

**專案簡介：**  
本專案為一套結合電腦視覺與深度學習技術的 AI 運動輔助系統。  
透過攝影機即時擷取使用者運動畫面，分析人體關鍵點位置，判斷姿勢是否正確，並即時給予動作修正建議，協助使用者提升訓練品質並降低運動傷害風險。

**使用技術：**
- 前端：HTML、CSS
- 後端：Python（Flask）
- 電腦視覺：OpenCV、MediaPipe
- 資料庫：MySQL

**系統功能：**
- 即時人體關鍵點偵測
- 深蹲 / 伏地挺身 / 仰臥起坐姿勢判斷
- 動作角度計算與錯誤提示
- 訓練次數與歷史紀錄儲存
- 使用者登入與資料管理

**負責內容：**
- 系統架構設計與流程規劃
- 使用 OpenCV 與 MediaPipe 建立姿勢辨識模型
- 動作角度計算與判斷邏輯設計
- MySQL 資料庫設計與串接
- 前端畫面設計與 API 串接
- 系統測試與效能優化

**成果：**
- 成功達成即時姿勢偵測（延遲低於 0.5 秒）
- 提升姿勢辨識準確率至 90% 以上
- 完成可實際操作之 Web 系統展示
